# Managing state with React Query. 〽️

This time, we are going to learn how to use React Query as a state management. 🧐

![cover](./src/assets/cover.png)

&nbsp;

## **Demo app features ⚙️**

1. list users from API.
1. create new users.
2. update username 
3. delete user by id.

&nbsp;

## **Technologies 🧪**

- React JS **18.2.0**
- React Query **4.20.4**
- React Router Dom **6.6.1**
- TypeScript **4.9.3**
- Vite JS **4.0.0**
- CSS vanilla

&nbsp;

## **Installation 🧰**

1. Clone the repository (you need to have [Git](https://git-scm.com) installed).

```shell
    git clone https://github.com/Franklin361/state-management-react-query.git
```

2.  Install dependencies of the project.

```shell
    npm install
```

3. Run the project.
```shell
    npm run dev
```
